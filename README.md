Program untuk menampilkan daftar pokemon

API endpoint :https://pokeapi.co/api/v2/pokemon/

Workflow pengerjaan
1. Membuat Database
2. GET data dari API dan INSERT ke database
3. Kemudian, data ditampilkan ke frontend dari database

Untuk menjalankan program, jalankan command pada terminal
"npm install express axios mysql2 cors"

run program dengan command
"node app.js"
